# DoggieCat
 
"DoggieCat" はPHPによる、犬と猫専用の画像共有SNSです。
 
# アプリケーション機能一覧
 
* ログイン機能
* 投稿機能
* コメント機能　
* フォロー機能
* 言い値機能
* ランキング機能
* カテゴリー機能
* 編集機能
* プロフィール機能
* Google Map API

 
# アプリケーション内使用技術

* AWS
* MySQL
* PHP/Laravel
* Javasctipt
* HTML
* CSS
* cloudinary
* heroku
* Google Map API
